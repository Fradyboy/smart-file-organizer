# Smart File Organizer (Python)

## Project Description
This Python automation tool organizes files automatically based on their file types. It scans a folder and moves files into categorized directories such as Documents, Images, Music, Code, and more.

This project reflects real-world automation commonly used for cleaning Downloads folders and managing office files.

------

## Features
- Automation file classification
- Folder creation if missing
- Safe file movement
- Easy customization

------

## Technologies Used
- Python 3
- OS module
- Shutil module
- File system automation
